## Tentang FlashCardStudy

FlashCardStudy adalah aplikasi belajar berbasis web untuk membuat, mengatur, dan mempelajari flashcard dengan cara yang sederhana dan interaktif.

Aplikasi ini membantu pengguna mengulang kosakata, melihat progres belajar, dan menjaga kebiasaan belajar tetap konsisten melalui mode kuis, flashcard, dan spaced repetition.

## Fitur

- Membuat, mengedit, menghapus, dan mengatur set flashcard.
- Mode kuis dengan pilihan jawaban dan ringkasan hasil belajar.
- Mode flashcard dengan flip, swipe, dan penanda kartu.
- Spaced repetition berbasis SM-2 sederhana.
- Login dan register untuk menyimpan set berdasarkan akun.
- Sinkronisasi set agar data tetap tersedia saat berpindah perangkat.
- Profil pengguna dengan statistik set, kartu, dan progres belajar.
- Public share untuk membagikan set flashcard.
- Tampilan responsif untuk desktop, tablet, dan mobile.
- Navigasi yang halus tanpa reload penuh antarhalaman.
